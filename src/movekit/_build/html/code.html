__init__.split_trajectories_fuzzy_segmentation(data_groups, segment=1, fuzzy_segment=2)¶

Split the trajectory of a single animal into several intervals (segments) according to some specific criterion.

Splitting may be interesting for example to detect different properties in time intervals. E.g. split into segments of 1 minute

Th fuzzy temporal segmentation is if the dataset is segmented into 10 min intervals, add a window of 2 minutes that overlaps on either side of the segments

Accepts: Python 3 dictionary containing ‘animal_id’ as key, and it’s Pandas Data Frame as value AND segments: an interval according to which the animals will be split into several Pandas Data Frames fuzzy_segment: an interval which will overlap on either side of the segments
